Top-Level Objects
^^^^^^^^^^^^^^^^^

These are the Page TSconfig Top-Level Objects (TLOs):


.. container:: table-row

   Property
         TCEMAIN

   Data type
         ->TCEMAIN

   Description
         Configuration for the TYPO3 Core Engine (TCEmain).


.. container:: table-row

   Property
         TCEFORM

   Data type
         ->TCEFORM

   Description
         Extra configuration for the form fields rendered by the TCEforms-class
         in general.


.. container:: table-row

   Property
         TSFE

   Data type
         ->TSFE

   Description
         Options for the TSFE frontend object.
